# 🏨 Hotel Data Analysis using Python

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on hotel booking datasets using Python and Pandas. The analysis includes data cleaning, transformation, occupancy analysis, revenue analysis, and generating business insights from hospitality data.

The complete project is implemented in a single Jupyter Notebook.

---

# 🎯 Objective

The objective of this project is to:

* Understand hotel booking datasets
* Perform data cleaning and preprocessing
* Analyze occupancy trends
* Analyze revenue patterns
* Generate meaningful business insights using data analysis techniques

---

# 🛠️ Technologies Used

* Python
* Pandas
* Jupyter Notebook

---

# 📂 Datasets Used

* dim_date.csv
* dim_hotels.csv
* dim_rooms.csv
* fact_aggregated_bookings.csv
* fact_bookings.csv

# 📊 Analysis Performed

The project includes:

* Data Exploration
* Data Cleaning
* Missing Value Handling
* Outlier Detection
* Occupancy Analysis
* Revenue Analysis
* City-wise Analysis
* Booking Platform Analysis
* Weekday vs Weekend Analysis
* Business Insights Generation

---

# 📈 Key Insights

Some important insights generated from the analysis:

* Average occupancy rate by room category
* Average occupancy rate by city
* Revenue realized by city
* Monthly revenue trends
* Property-wise revenue analysis
* Revenue contribution by booking platforms
* Weekday and weekend occupancy comparison
* Average ratings analysis by city
